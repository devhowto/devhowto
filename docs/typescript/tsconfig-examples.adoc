== Simplest tsconfig.json 1

Generated with `tsc` 5.3.3 on March 2024 (comments removed):

[source,json]
----
{
  "compilerOptions": {
    "target": "es6",
    "module": "commonjs",
    "strict": true,
    "checkJs": true,
    "esModuleInterop": true,
    "skipLibCheck": true,
    "forceConsistentCasingInFileNames": true,
  }
}
----

== Simplest tsconfig.json 2

Same as the above with `rootDir` and `outDir`:

[source,json]
----
{
  "compilerOptions": {
    "target": "es6",
    "module": "commonjs",
    "strict": true,
    "checkJs": true,
    "esModuleInterop": true,
    "skipLibCheck": true,
    "forceConsistentCasingInFileNames": true,
    "rootDir": "./src",
    "outDir": "./dist"
  }
}
----
